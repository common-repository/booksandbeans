adds a Books&beans - Button to generate ePub from the post 

== Description ==

This Pulgin renders the wordpress-Post Outputs, summits it into a formular to 
send it to booksandbeans.de where you can transfer it into the popular ePub-Format


A few notes about the sections above:


== Installation ==


1. Upload `booksandbeans`-Folder  to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
